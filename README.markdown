SpreeTestimonials
=================

Testimonials (guestbook) for Spree shop.


Installation
=======

Add to Gemfile:
    gem 'spree_testimonials', :git => 'git://github.com/divineforest/spree_testimonials.git'

and then run
    bundle install
    rake spree_testimonials:install && && rake db:migrate
